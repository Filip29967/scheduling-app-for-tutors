## General info
Desktop application for tutors, that can be used to manage lesson schedule. Allows creating list of students with assigned date and time.


## Features
- Add new students with date and time 
- Display list of added students
- Save database with students locally
## Planned Features
- Design GUI
- Edit students data
- Login to admin account
- Calendar view
## Technologies
- Java 21
- JavaFX 21
- SQLite
- Maven
- Scene Builder
## Setup
Requirements: Java 21, git<br/>
Clone the repo
```git clone https://github.com/Filip29967/scheduling-app-for-tutors.git``` <br/>
Change directory ```cd scheduling-app-for-tutors```<br/>
Run app with maven ```./mvnw clean javafx:run```<br/>
